The Paul McCartney concert last night in Tel Aviv was broadcast on Reshet Bet and I wanted to listen online. But the IBA Internet Broadcast Centre hasn't been designed with any thought for cross-platform standards-compliance. They use the proprietary .asx format and Windows Media codecs. Because of this incompetence, they are forcing users to use Microsoft Windows.  I'm not sure if it works on Mac OS, it certainly doesn't work on my Linux system (update: I have had some success with mozplugger but it is painful).  I dived into their setup and found the real URLs, which work with any operating system.

You'll need a media player and the relevant codecs installed, I use mplayer but I'm sure others e.g. VLC or Totem will work as well. If your system is configured correctly, your web browser will open the link in your media player, and the stream will start:

  • Reshet Aleph: ?
  • Reshet Bet: mms://s53wm.castup.net/990310001-52.wmv
  • Reshet Gimel: mms://s52wm.castup.net/990310004-52.wmv
  • Reshet Dalet (Arabic):  mms://s53wm.castup.net/990310003-52.wmv
  • Reshet Moreshet: mms://s52wm.castup.net/990310002-52.wmv
  • Reshet Klitat Aliyah (REKA): mms://s36wm.castup.net/990310005-52.wmv
  • Kol Hamusica: mms://s53wm.castup.net/990310008-52.wmv
  • 88FM: ?

If your web browser isn't configured correctly, you can still open the stream from the shell. For example, to listen to Reshet Bet, just type:

mplayer mms://s53wm.castup.net/990310001-52.wmv
